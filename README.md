# Employee-Management-System
Employee Management System - A full-stack CRUD application for managing employee records with Spring Boot (Java) backend and React (Vite) frontend. Perfect for small to medium-sized organizations to maintain employee data.

##  Features

## Employee Record Management

- **Add New Employees** - Register employees with name, phone, department, and position details
- **View All Employees** - Display all employees in a responsive table with serial numbers
- **Update Information** - Edit existing employee details seamlessly
- **Delete Employees** - Remove employees with confirmation dialog

## Technical Highlights

- **Automatic Timestamp** - Entry date/time automatically recorded when employee is added
- **Real-time Updates** - Changes reflect instantly in the UI without page refresh
- **Responsive Design** - Works flawlessly on desktop, tablet, and mobile devices
- **RESTful API** - Clean separation between frontend and backend
